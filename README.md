# React 샘플 프로젝트 (백엔드 firebase 활용)

react 샘플 프로젝트 소스 코드입니다.

## 기술 스택

- React.js 17.0.2
- React router
- use-context
- Redux
- Axios
- Firebase

## 개발 환경

- [Git](https://git-scm.com/downloads)
- [Visual Studio Code](https://code.visualstudio.com/)
- [Node.js LTS 버전(v14.x 이상)](https://nodejs.org/ko/)
